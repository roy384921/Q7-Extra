# 本次模擬之五個不同場景
## 1. 場景一：全區隨機分佈 (Randomly distributed)
* 空間位置：使用者隨機出現於整個-20到20的區域內（X 與 Y 座標皆介於 -10 到 10 之間）。
* 移動狀態：以每秒 0.5m 的速度，朝 X 與 Y 的正向斜角緩慢移動。
* 特徵意義：作為最基礎且多樣化的通道資料，涵蓋了各種距離與角度的訊號衰落。
## 2. 場景二：近中心叢集 (Clustered near the center)
* 空間位置：使用者集中在基地台附近，分佈於半徑約 2.5m 的核心區域內。
* 移動狀態：移動極為緩慢 (v = [0.1,-0.1,0]），近乎靜止。
* 特徵意義：模擬使用者距離基地台極近、具有強烈視距 (LOS) 訊號且延遲較小的靜態環境。
## 3. 場景三：邊緣使用者 (Edge users)
* 空間位置：使用者嚴格分佈在 20*20m 區域的「邊界」上（座標為 10 或 -10）。
* 移動狀態：以每秒 1m 的速度，純粹沿著 X 軸方向移動。
* 特徵意義：模擬距離基地台最遠的極端情況，訊號衰減最大，且可能伴隨較嚴重的多徑干擾 (Multipath interference)。
## 4. 場景四：特定斜向快速移動 (Fast moving diagonally)
* 空間位置：使用者分佈於 15*15 m 的較大範圍內。
* 移動狀態：以每秒 2m 的相對高速，朝著對角線方向（v = [2.0, 2.0, 0]）快速移動。
* 特徵意義：這會產生較顯著的都卜勒頻移 (Doppler shift)，考驗通道狀態在時間上的快速變化性。
## 5. 場景五：全區靜止 (Stationary users)
* 空間位置：使用者同樣隨機分佈於整個 20*20 m 的區域內。
* 移動狀態：完全靜止不動（v = [0, 0, 0]）。
* 特徵意義：建立純靜態的通道環境，用以對照其他移動場景，觀察純粹由空間位置差異帶來的角度與延遲特性變化。
# 模擬結果與分析
## 單一場景訓練的侷限性（b小題：面臨 Domain Shift）
* 在b小題中，我們使用僅在單一基礎場景下訓練的 CsiNet 模型，去推論五個具備不同使用者分佈與移動速度的全新測試集。
* 效能顯著劣化：測試結果顯示，當面臨未曾見過的通道特徵時，模型的重建表現（NMSE）並不如預期理想（例如誤差停滯在較高的水平，-1~-2 dB 附近）。
* Domain Shift效應: 這個現象證明了資料驅動 (Data-Driven) 模型的脆弱性。由於訓練資料缺乏多樣性，神經網路會過度擬合（Overfit）於單一環境的局部特徵，一旦實際物理環境改變（如使用者移至邊緣或高速移動導致不同的角度-延遲域分佈），編碼器便無法有效萃取通道結構，導致解碼器重建出失真或帶有大量雜訊的 CSI。
## 混合場景訓練的強健性（c小題: Generalization 能力的提升）
* 在c小題中，我們將涵蓋多種極端分佈（如近中心、邊緣、高速移動等）的五個資料集混合，重新訓練了一個具備廣泛視野的 CsiNet 模型。  
* NMSE 的全面改善：與b小題的結果形成強烈對比，經過混合訓練後，模型在各個獨立測試集上的 NMSE 皆有顯著幅度的下降（改善）。這證明神經網路成功適應了複雜的動態環境變化。
* 通用特徵的學習 (Universal Feature Representation)：這顯示當賦予足夠多樣的訓練樣本時，CsiNet 能夠突破單一場景的限制，學習到角度-延遲域中更深層、更通用的物理稀疏結構，而非死記硬背特定位置的通道樣貌，從而進行了有效的非線性特徵壓縮與補償。
